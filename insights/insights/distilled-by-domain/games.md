### 🔺 情報非対称性CP理論（2026-04-15 新規）

- **ゲームCP条件 = 情報格差の最適管理**: 完全情報（退屈）↔ 完全無情報（混乱）の間のCP。Among Us / Dead by Daylight はこの設計の精華 → [[threads/2026-04-15_game-fun-information-asymmetry-cp-theory]]
- **解決されたゲーム = Sym-Frozen問題**: メタ固定化 = 情報格差ゼロ化 = Sym-Frozen。パッチ = MEA注入（情報格差の人為的復元）→ 同上
- **サハロフ3条件のゲーム適用**: 戦略非保存 + 情報非対称 + 非Nash均衡 = エンゲージメント必要条件（宇宙バリオン生成・PCE認知変容と3条件で同型）→ 同上
- **ゲームの時間的CP = 情報動態の1/f**: 面白いゲームは情報が複数タイムスケール（秒/分/試合全体）で展開 = 1/f的情報構造 → 同上
- **ゲームデザイナー = 情報非対称性エンジニア**: ゲームディレクター = プレイヤーPrior Collapse Engineer。パッチ = メタ（π_prior）崩壊 + 新情報格差（π_likelihood）注入 → 同上
- **フロー理論 = PCC特殊ケース**: スキル≈チャレンジ = π_prior≈π_likelihood = PCC。面白さの3次元CP空間 = フロー軸 + 情報格差軸 + 1/f軸 → 同上
