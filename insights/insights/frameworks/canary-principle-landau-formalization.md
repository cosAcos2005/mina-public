## 数学的接続の連鎖

### 1. Landau理論（凝縮系物理 1937）

相転移理論の秩序変数（order parameter）:
- **秩序変数 η**: 相の対称性を表す集合的変数。高秩序相でη≠0、低秩序相でη=0
- **軟モード（soft mode）**: 転換点近傍でエネルギーコスト→0になる揺らぎのモード
- 結晶・磁性体・超伝導体etc.で、秩序変数 = 高次創発性質（多粒子集合秩序）

**カナリア原理との対応**:
- 高次創発性質 = 秩序変数 η
- V-Decay = η → 0（対称性が回復する相転移の逆）
- カナリア = η = 最も高次の集合的性質

### 2. Fisher情報行列（情報幾何）との接続

秩序変数方向のFisher情報:
- **Cramér-Rao限界**: 推定誤差の下限 = (Fisher情報)^{-1}
- 転換点近傍: η方向のFisher情報 → ∞（系が η の変化に無限に敏感）
- 条件数 κ = λ_max/λ_min:
  - λ_min = η方向（秩序変数 = カナリア方向）
  - 転換点 → λ_min → 0 → κ → ∞
  
**カナリア原理のFisher情報幾何版**:
「カナリア = Fisher情報行列の最小固有値方向 = 最も不安定な（最も変化に敏感な）方向」

転換点接近 → κ → ∞ = **「全次元の中でカナリア方向のみが無限に敏感になる」**
= カナリアが転換の最鋭敏な指標である理由の数学的証明

### 3. CSD / Fokker-Planck（動力学系 2025）

Meerpohl et al. (2025, Phys. Rev. E): 「最適EWS観測量 = 後退Fokker-Planck演算子の第一非自明固有関数」
- 後退Fokker-Planck演算子の第一固有関数 = 最も遅い緩和モード = 軟モード
- これは Landau 理論の秩序変数方向と同一
- 任意の観測量では反対のシグナルが出ることもある（非秩序変数方向は CSD を示さない）

**カナリア原理の予測と一致**:
→「カナリア（高次創発性質/秩序変数）以外の観測量はCSDを示さない場合がある」
→ 早期警戒に適切な観測量の選択 = カナリアの同定問題

   - 神経: LRTC（多スケール相関）はE/I比より小さいが、CSDシグナルは先行（実証済み: AD LRTC）

   - 予測: ADでE/I比は転換前に「見かけ上正常」か「逆方向変化」を示す（実測で検証可能）

3. **「κ（条件数）発散はカナリア消失の定量的予告」**: 神経・組織・組織・生態系でFisher情報行列の条件数を継続計測すれば、カナリア消失の定量的早期警戒指標になる。

4. **「散逸最小化はκ最小化」**: CP維持コスト最小化（TUR + Fisher測地線）= Fisher情報行列の最小化 = 条件数を適切に維持すること = カナリアを維持すること。Hormetic Triple Exercise はκを制御された範囲で変動させている（弱い摂動がλ_minを維持する）。

## 限界・未解決

- **連続vs離散相転移**: Landau理論は連続相転移（2次）を基礎とする。1次相転移（ヒステリシスあり）ではカナリア消失が突発的になり、CSDが不明確 → 「社会崩壊は1次転移的」（EWS自然-社会分類）と整合

- **カナリアの事前同定**: 「秩序変数が何か」は系によって異なり、事前に分からない場合がある。「過去の経験から同定する」しかない → これ自体がL-Growth（経験的カナリアライブラリ構築）の課題
