## 💡 新指標 D = α - (2H-1) — 「fGn からの偏差」

### fGn 理論ラインの意味

```
Fractional Gaussian Noise（fGn）の理論:
  α = 2H - 1

  この式が意味すること:
  「完全に自己相似な確率過程では、
   スペクトル指数（α）と Hurst 指数（H）は一対一対応する」
  
  = 「周波数構造（α）と長距離相関（H）が整合した状態」
  = 理論的に最も「ピュア」な自己相似ランダム過程
```

### D の物理的意味

```
D = α - (2H-1) = α からの fGn 予測の偏差

D >> 0（α-rich）:
  = 「H から予測されるより α が大きい」
  = 「スペクトル的に過剰構造化されている」
  = Verlet（D≈+5.5）: 決定論的振動が支配 → 低周波に過剰エネルギー
  = 単純 Langevin（D≈+3.0〜3.7）: Lorentzian ピーク → 低周波偏重

D ≈ 0（fGn-like）:
  = 「α と H が fGn 理論と整合」
  = 「自然な多スケールランダム過程」
  = 生体信号・200個 Lorentzian・fGn H=0.9

D < 0（H-rich）:
  = 「α から予測されるより H が大きい」
  = 「スペクトルより長距離相関が強い」
  = SOC（D≈-0.38）: 疎な離散事象列 → α が低いが H は高い
  = fGn H=0.6〜0.7（D≈-0.1〜0）: ほぼ fGn 上だが若干 H-rich
```

> [!important] 発見
> **D は「系のダイナミクスの性質（決定論的 vs 確率的 vs 離散事象型）」を定量化する新指標。**
> - D >> 0: 「構造に支配された系」（Frozen 的）
> - D ≈ 0: 「スケールフリーな確率過程」（CP 的）
> - D < 0: 「事象駆動的な系」（SOC 的）

## 🎯 刷新された CP ゾーン定義

### 旧定義（1次元）

```
旧: α ≈ 1 ならば Coherent Phase
問題: SOC（α≈0, H=0.7）が除外される
     fGn H=0.7（α=0.4）が除外される
     Verlet-like な α=1 偶然一致が混入する可能性
```

### 新定義（2次元）

```
新: Coherent Phase = 以下の3条件を全て満たす
  ① 0.3 ≤ α ≤ 1.5  （スペクトルにべき乗則構造がある）
  ② 0.60 ≤ H ≤ 0.88 （長距離相関がある、過剰ではない）
  ③ |D| = |α - (2H-1)| ≤ 0.6  （fGn 線から大きく外れない）

補足カテゴリ:
  SOC型CP（H-rich CP）: α < 0.3, H ∈ [0.60, 0.75], D < -0.2
  = SOC のような疎事象システム

除外ゾーン:
  Frozen: α > 3.0 or H > 0.97
  Pure Disordered: α < 0.3, H < 0.55
  Multi-scale Dissipative: α ∈ (1.5, 3.0), H ∈ (0.85, 0.97)
```
