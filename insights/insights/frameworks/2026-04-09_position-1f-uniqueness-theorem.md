# 位置 1/f 生成の唯一性定理

> [!abstract] 概要
> session13 Cycle5〜session14 Cycle8 にわたる実験群から、数学的に明確な命題が浮かび上がった: **「バネ系における位置の 1/f スペクトルは、多スケール Langevin 散逸の存在が必要十分条件である」。** これは「位置の 1/f 生成の唯一性定理」として定式化できる。系統的に試した全てのアプローチ（孤立系/位置ノイズ/FPU/閾値SOC/乾性摩擦SOC）が全て失敗し、多スケール Langevin のみが成功した事実が、この定理を経験的に強く支持する。

## 📐 定理の定式化

### 命題（経験的に確立）

```
定理（位置 1/f の唯一性）:

  n 個の結合バネ系において位置 x_i(t) のパワースペクトル密度が
  
      S(f) ∝ f^{-α}, α ∈ (0.3, 1.5)  （CP ゾーン）
  
  を k 桁以上の周波数帯域にわたって満たすとき、
  その系は「k 桁以上の時定数分布を持つ多スケール散逸機構」を
  必然的に含む。
  
  等価に:
  「多スケール散逸（多スケール Langevin）は位置 1/f の
   必要条件であり（実験的に強く示唆）、十分条件である（Cycle9 で証明）。」
```

### 証明のスケッチ（十分条件の方向）

```
【なぜ多スケール Langevin が 1/f を生むか】

OU 過程（単一時定数 τ）のパワースペクトル:
  S_OU(f, τ) ∝ τ / (1 + (2πfτ)²)  — Lorentzian
  
  f << 1/τ: S ≈ τ（定数プラトー）
  f >> 1/τ: S ≈ 1/(4π²τ f²) ∝ f^{-2}

多スケール OU（τ が対数一様分布 [τ_min, τ_max]）の合成スペクトル:
  S(f) = ∫ S_OU(f, τ) d(log τ)
       = ∫ 1 / (1 + (2πfτ)²) dτ / (ln(τ_max/τ_min))
       = arctan(2πf·τ_max) - arctan(2πf·τ_min)
         ─────────────────────────────────────────
               2πf · ln(τ_max/τ_min)
               
  1/τ_max << f << 1/τ_min のとき:
    arctan(2πf·τ_max) → π/2, arctan(2πf·τ_min) → 0
  → S(f) ≈ π / (4f · ln(τ_max/τ_min)) ∝ 1/f  ✓

  これは π/(4·ln(τ_max/τ_min)) 桁の周波数帯域で 1/f を実現する。
  
  「多スケール Langevin → 位置 1/f」 ← 十分条件 ✅
```

### 必要条件の証拠（実験的）

```
系統的な「失敗の証拠」= 多スケール散逸なしに 1/f が出ない実証

系統  | 実装 | α | 理由
────────────────────────────────────────────────────────
孤立系     | Pure Verlet             | ≈6.4 | エネルギー保存 → 散逸なし

──────────────────────────────────────────────────────
```

## 💡 なぜ他の機構が失敗するか — 統一的説明

```
全ての失敗機構に共通するパターン:

【失敗パターン 1: 「積分の法則の壁」】
  位置に白色ノイズ → バネが積分 → α≈2（Random Walk）
  位置に 1/f ノイズ → バネが積分 → α≈4（さらに悪化）
  加速度に 1/f ノイズ → 2重積分 → α≈5〜6（最悪）
  
  = 「何らかのノイズを注入しても、バネ力が積分して
     スペクトル指数を増大させる」

【失敗パターン 2: 「単一タイムスケールの壁」】
  単一 Lorentzian: S(f) ∝ 1/f² (f >> 1/τ)
  = 「1つのタイムスケールしかない → α≈2 or 4、1/f にならない」

【失敗パターン 3: 「位置と事象の分離」】
  SOC 系: 事象列 α≈0, H≈0.66（1/f 的）
          位置   α≈6（Frozen）
  = 「SOC の 1/f は事象に現れるが位置には現れない」
  = 「視覚的な 1/f には位置の多スケール散逸が必要」

多スケール Langevin が成功する理由:
  複数の独立した Langevin 熱浴が並列動作
  → 各スケールの Lorentzian ピークが重なる
  → 対数一様な τ 分布 → 積分により S(f) ∝ 1/f
  = 「積分の法則を逆用: 多数の Lorentzian を並べてその合成が 1/f になる」
```
