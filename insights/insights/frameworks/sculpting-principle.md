# 彫刻の原理（Sculpting Principle）

## 一言で

**可能性空間を適度に削ることで、残った空間に構造が自然と浮かび上がる。**
削りすぎると壊れる。削らなすぎると何も見えない。最も豊かな形は、臨界点の直前に���る。

## 構造

```
[可能性空間（何でもありの状態）]
      ↓ 制約を加える（＝可能性を削る）
[解空間が縮小し、クラスタが形成される]
      ↓ さらに制約を加える
[クラスタ間に構造・パターンが浮かび上がる] ★ここが最適点
      ↓ 制約を加えすぎる
[解空間が消滅。構造が壊れる]
```

## 根拠（4分野の統合）

同じ構造が全く異なる分野で独立に発見されている：

| 分野 | 現象 | 制約のメカニズム |
|-----|------|---------------|
| 認知科学 | 制約→第一直感の無効化→探索の強制 | Haught-Tromp (2016), Acar et al. (2019) メタ分析145本 |
| CSP理論 | 制約→解のクラスタ化→臨界点で最も豊かな構造 | ランダムCSPの相転移（α_sat付近） |
| 物理シミュ | 制約の干渉→アトラクター形成 | PBD/XPBDの制約ソルバー |
| 生物学 | 活性化+抑制の2制約→パターンの自己組織化 | チューリングパターン（1952） |

**共通する本質**: 制約が可能性を削ることで、残った空間に構造が浮かび上がる。

## 逆U字カーブ（最重要）

```
構造の豊かさ
    ↑
    │         ★ 臨界点（最も豊か）
    │        / \
    │       /   \
    │      /     \
    │     /       \
    │____/         \___
    ─────────────────→ 制約の量
  少ない（自由すぎ）    多すぎ（過拘束）
```

- **制約ゼロ**: 自由すぎて構造なし（解がバラバラ / 人は第一直感を選ぶ）
- **制約が臨界付近**: 最も豊かな構造（解がクラスタ化 / なじみの薄い道を探索）
- **制約が多すぎ**: 解なし（過拘束 / 人は潰れる）

## フラク��ル分解との対関係

| | フラクタル分解 | 彫刻の原理 |
|---|---|---|
| **問い** | 「これは何でできてるか？」 | 「これをどう作るか？」 |
| **方向** | 構造を分解して理解する | 制約を加えて構造を浮かび上がらせる |
| **用途** | **分析** | **生成** |

**分析と生成のペア**。フラクタルで理解し、彫刻で創る。

## 使いどころ

| 場面 | 適用 |
|-----|------|
| アイデアが出ない時 | 「自由に考えて」→「この3つの制約の中で考えて」に変える |
| 髪物理のパラメータ調整 | stiffnessは「臨界点の直前」を狙う |
| ES・ポートフォリオ | 字数制限・形式制約を**味方**として使う |
| プロンプト設計 | 制約を1-3個入れると出力の質が上がる |
| ゲームデザイン | ハードウェア制約を積極的に受け入れてメカニクスを発明 |
| 目標設定 | 「何でもできる」より「この期限でこの範囲で」 |

## 制約の質マトリクス

すべての制約が平等ではない：

| 制約の種類 | 効果 |
|-----------|------|
| 技術的制約（RAM、時間、予算） | 創造的ワークアラウンドを強制 → ✅ ポジティブ |
| 目的的制約（「このユーザーに届ける」） | 思考の焦点化 → ✅ ポジティブ |
| 官僚的制約（「このフォーマット以外禁止」） | 自律性を侵害 → ❌ ネガティブ |
| 矛盾する制約（AかつnotA） | 認知負荷の飽和 → ❌ 壊滅的 |

## 限界

- 最適な制約の量は文脈依存。普遍的な数値はない
- 「どの制約を加えるか」の選択自体に創造性が必要（メタ制約問題）
- 制約を「脅威」と認知すると逆効果（Rosso, 2014）。「チャンス」と解釈できる環境が前提
